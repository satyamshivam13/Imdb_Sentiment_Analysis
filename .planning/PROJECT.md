# IMDb Sentiment Analytics Platform

## What This Is

A Flask-based sentiment analytics platform that extends single-review inference into a complete analytics narrative for internal stakeholder demos.

## Core Value

Internal stakeholders can quickly understand sentiment trends and model quality from review data, not just single-review outputs.

## Current State

- v1.0 shipped on 2026-04-02.
- Delivered capabilities: persistence foundation, batch CSV analysis, dashboard visualization, metrics observatory, and history experience hardening.
- Documentation reconciliation completed in Phase 6 to align governance artifacts with implementation evidence.

## Shipped in v1.0

- Persistent history capture across UI/API/batch inference flows.
- Batch CSV validation, mixed-result scoring, report views, and export downloads.
- Dashboard charts for sentiment distribution and trend with date-range controls.
- Metrics surface for summary scores, confusion matrix, and model metadata.
- History browse and clear flows with regression coverage.
